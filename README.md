APLICACION DE COMIDAS RAPIDAS
Para iniciar la aplicacion tiene que realizar los siguientes comandos:

cd fastfood
composer install
php artisan migrate

y se ejecuta la aplicacion de la forma que mejor desee, ya sea utilizando laragon o xampp.
en el caso de laragon reinicia los sevicios de laragon y entra a la app en el caso de usar xampp

 npm install && npm run dev
 php artisan serve
